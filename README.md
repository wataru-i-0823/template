# Template

## フォルダ構造
```
.
└── template
    ├── README.md 本ファイル.
    ├── data ファイル格納用ディレクトリ.
    │   ├── 00_raw 生データ格納用ディレクトリ.
    │   ├── 01_interim 整形後のデータもしくは中間データ格納用ディレクトリ.
    │   ├── 02_dataset データせっと格納用ディレクトリ.
    │   ├── 10_model モデルファイル格納用ディレクトリ.
    │   └── 99_other その他ファイル格納用ディレクトリ.
    ├── docs ドキュメント格納用ディレクトリ.
    ├── notebook ノートブック格納用ディレクトリ.
    ├── reports 結果格納用ディレクトリ.
    │   ├── 01_base_agg 基礎集計格納用ディレクトリ.
    │   └── 02_model モデルファイル格納用ディレクトリ.
    ├── src モジュール格納用ディレクトリ.
    │   ├── data
    │   ├── evaluate
    │   ├── model
    │   ├── utils
    │   └── visualize
    └── work 実行用pyファイル用ディレクトリ.
        └── config 設定ファイル用ディレクトリ.
    ```
